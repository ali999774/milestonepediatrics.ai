## What this is
Full website build for a new solo, insurance-based general pediatric practice
opening in Houston, TX in approximately 6 months. Built using the Agentic Build
Studio methodology — a simulated professional studio where each AI agent has a
named identity, scoped files, and a complete brief.

## Stack
React + Vite + TypeScript · Tailwind CSS · React Router v6 ·
Supabase (waitlist form) · Vercel (hosting) · Namecheap (DNS)
